# 目录结构说明

```

│   ├─ assets 静态资源目录
│   ├─ uploads 上传目录
│   ├─ favicon.ico ICON图标文件
│   └─ index.php 入口文件
├─ storage 缓存目录

    # vendor                         第三方类库目录
```
